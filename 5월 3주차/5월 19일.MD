# Review
### MVC란?
- Model : 비즈니스 로직처리
- View : 프리젠테이션 로직처리
- Controller : 사용자와의 인터페이스, 전체 프로그램의 흐름제어 담당


분리함으로써 <u>***유지보수***</u> 와 <u>***코드 재사용***</u> 을 쉽게 하기 위해 만들어진 디자인 패턴이다.

### 구조
```
사용자 -> 컨트롤러 -> 모델  ->
   |         |            <-    DB
   ㅡ    프리젠테이션

```
