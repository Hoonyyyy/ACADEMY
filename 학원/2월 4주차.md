### print
출력(print)
- 1) println() : 줄바꿈
- 2) print()	 : 줄바꿈 안됨
- 3) printf("저의 이름은%s이고 나이는 %d입니다", "홍길동", 20)	 : 변환이 필요할때?

~~~java
public class Print1 {
	public static void main(String[] args) {
		
		// print
		System.out.print("hi");
		System.out.print("34345hihihihdfdfdfihi");
		
		// println
		System.out.println("자바자바");
		
		// printf
		System.out.printf("제 이름은 %s이고 나이는 %d 입니다", "후니", 20); // %.4f 소수점 4자리
	}
}
~~~
