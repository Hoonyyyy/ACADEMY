### 하이퍼링크
```html
<!DOCTYPE html>
<!-- 
	하이퍼링크
		<a href="URL | URL#name" target="윈도우" > 이동할위치 </a>

			- target :
				. _blank: 새창에 출력
				. _self : 현재창에 출력
				. _parent: 부모창에 출력
				. _top: 최상위에 출력

		. 이동
			- 다른 페이지
			- 같은 페이지의 특정위치
			- 다른 페이지의 특정위치
			- 사이트 이동
		. 출력
			- 내 자신에게 출력
			- 부모에게 출력

 -->

<html>
<head>
<meta charset="UTF-8">
<title>하이퍼링크추가</title>
</head>
<body>
	<a href="https://www.naver.com/">네이버</a><br>
	<a href="https://www.youtube.com" target="_blank">유튜브</a><br>
	<a href="https:/www.daum.net" target="_blank"> 
		<img src="./img/flower.jpg" width="50" height="50">
		다음사이트로 이동</a>

</body>
</html>
```
